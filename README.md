# flowers

微信小程序：一个繁花曲线


## 概述

本项目是繁花曲线在微信小程序上的一个实现。

你可以通过调整「子尺半径」「绘点半径」和「绘制颜色」等参数来绘制各种各样的繁花曲线图，同时支持设置背景图、保存到相册、发送给好友。
